## How to start

To install and connect the RouteLink/Client, follow these steps:

1. Clone the repository:

    ```bash
    git clone https://github.com/routelink/client.git
    ```

2. Install dependencies:

    ```bash
    npm install
    ```

3. Launch the application:

    ```bash
    npm start
    ```
